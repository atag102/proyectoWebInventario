*Nombre                                                                                                                                   
*Apellido                                                                                                                                 
*CI(cedula de identidad)
*Fecha de nacimiento                                                                                                                       
*Direccion                                                                                                                                 
*Correo electronico                                                                                                                       
*telefono                                                                                                                                 
*username
